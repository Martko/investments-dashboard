# investments-dashboard

service provides an API for `investments-dashboard-client`.

## Stack

- Node.js
- TypeScript
- MySQL

## Installation

1. Create configuration file based on example.

   ```
   $ cp .env.example .env
   ```

2. Install required modules

   ```
   $ npm install
   ```

## Development

```
$ npm run watch-ts
```

### Debug

```
$ npm run serve-debug
```
